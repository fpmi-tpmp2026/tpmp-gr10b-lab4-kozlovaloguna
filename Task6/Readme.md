# Цветочная оранжерея

## Описание проекта

**Цветочная оранжерея** — это автоматизированная система для управления заказами цветочной мастерской. Оранжерея выращивает различные виды цветов и создаёт из них композиции на заказ. Проект позволяет вести учёт цветов, композиций, заказов и покупателей, а также рассчитывать итоговую стоимость с учётом срочности выполнения (надбавка 25% за 1 день, 15% за 2 дня). Система предоставляет аналитические отчёты по продажам, спросу на композиции, использованию цветов и срочности выполнения заказов. Дополнительно реализованы функции контроля ценообразования и автоматического расчёта стоимости заказа.

## Структура базы данных

- **Цветы** — название, сорт, стоимость одного цветка.
- **Композиции** — название композиции, цветок, сорт, количество единиц.
- **Заказы** — дата принятия, дата выполнения, композиция, количество, покупатель.
- **Покупатели** — информация о клиентах.

## Функциональность

1. **Создание таблиц БД** с ограничениями целостности.
2. **Выдача аналитической информации** с помощью `SELECT`:
   - Сумма полученных денег за указанный период.
   - Композиция с максимальным спросом (все сведения).
   - Количество выполненных заказов по срочности.
   - Количество использованных цветов по видам и сортам за период.
   - Количество проданных композиций и сумма денег по видам композиций.
3. **Обновление данных** через `INSERT`, `UPDATE`, `DELETE`.
4. **Функция запрета повышения цены** на цветы, если стоимость композиции возрастает более чем на 10%.
5. **Функция внесения заказа** — автоматический расчёт стоимости и сохранение в специальную таблицу.
6. **Функция вывода информации** о полученных заказах на указанную дату.
